2. TailwindCSS 설치

TailwindCSS 설치를 위해 아래 명령어를 실행합니다.

npm install -D tailwindcss
npx tailwindcss init

명령어를 차례대로 실행하였다면 tailwind.config.js 파일이 생성됩니다.

tailwind.config.js 파일을 아래와 같이 수정합니다.

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ["./src/**/*.{html,js}"],
  theme: {
    extend: {},
  },
  plugins: [],
}
